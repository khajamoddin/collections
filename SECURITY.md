# Security Policy

## Supported Versions

The main branch is supported. Released tags are supported until the next minor release unless marked otherwise.

## Reporting a Vulnerability

- Email: `khaja@estimatedstocks.com`
- Or open a private GitHub Security Advisory for this repository.

Please include a minimal reproduction if possible. You will receive an acknowledgment within 3 business days. We strive to provide a fix or mitigation timeline within 14 days, depending on severity.

## Handling

1. Reports are triaged privately.
2. Fixes are developed and tested in a private branch.
3. Coordinated disclosure timelines are discussed with the reporter.
4. A security release and changelog entry are published when ready.

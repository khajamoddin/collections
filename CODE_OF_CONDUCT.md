# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

- Be respectful and inclusive.
- Welcome new contributors and diverse viewpoints.
- Give and gracefully receive constructive feedback.
- Resolve disagreements with empathy; escalate when needed.

## Reporting

If you experience or witness unacceptable behavior, please report it to the maintainers at `khaja@estimatedstocks.com` or via a private GitHub Security Advisory. Reports are handled confidentially and promptly.
